compound operators

compound assignments are shorter and therefore simpler (less complex) than using regular operators

compound operators are

  1. plus equalsto(+=)
  2. minus equalsto(-=)
  3. multiply equalsto(*=)
  4. division equalsto(/=)

comparision operators

Comparison operators are used to compare two values (or variables). This is important in programming, because it helps us to find answers and make decisions. The return value of a comparison is either 1 or 0 , which means true (1) or false (0).

types of comparision operators

  1. lessthan => if the statement is crct it will show true otherwise it will show false
  2. greaterthan => if the statement is crct it will show true otherwise it will show false
  3. lessthan or equalsto => any one statement is crct either lessthan or equal it will show true otherwise it will show false
  4. greaterthan or equalsto => any one statement is crct either greaterthan or equal it will show true otherwise it will show false
  5. double equalsto => lhs equals to rhs -> compare two variables ignore datatype
  6. not equalsto => compare two variables with datatype

logical operators

logical operators are used to determine the logic between variables and values

  1. AND(&&)=>both operations are true it will show true otherwise not
  2. OR(||) =>one operation is true among two it will show true otherwise not
  3. NOT(!) =>opposite -> true means it will show false,false means it will show true
